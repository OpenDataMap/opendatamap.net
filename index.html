Was ist die OpenDataMap?

Worum geht's?

Die OpenDataMap ist eine modular aufgebaute Kartenlösung fÃ¼r unterschiedlichste offene Datenformate. Der unterschied zu anderen Karten ist, dass auf der OpenDataMap die lokalen Daten aus mehreren Quellen zusammengefasst sind, sodass ein interessierter Bürger für seine*ihre lokalen Daten nur noch eine Website braucht.

Wie kann ich mitmachen?

Super, dass du mitmachen möchtest! Du kannst dich auf verschiedenste Arten einbringen.

Du willst:

  • mitprogrammieren
    Wir können immer Leute brauchen, die mit uns weiterentwickeln! Wenn du einen Bug gefunden hast oder dir ein Feature wünschst kannst du gerne in unserem GitHub ein Issue oder noch besser einen Pull Request erstellen. Wenn du aus der Region Bonn/Rhein-Sieg kommst kannst du gerne mal zu einem Meetup im OK Lab Bonn/Rhein-Sieg kommen. Bei sonstigen Fragen kannst du auch gerne an kontakt (ät) opendatamap.net schreiben.
  • eine eigene Karte betreiben
    Super dass du eine Karte betreiben willst, am besten kontaktierst du uns via E-Mail an kontakt (ät) opendatamap.net um dir bei der Einrichtung zur helfen.
